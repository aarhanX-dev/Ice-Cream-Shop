# 🍦 Smart Ice Cream Shop (CLI Python Project)

**Author:** Aarhan (`Shadow dev`)  
**Language:** Python 3  
**Project Level:** Beginner → Intermediate  

---

## 📝 Project Description

This is a simple **Command Line Interface (CLI) Ice Cream Shop simulator** built in Python using **OOP concepts**.  

Users can:

- View the **menu of ice creams and toppings**.  
- Choose a **flavour** and optional **toppings**.  
- See the **bill** for their ice cream order.  
- Handle **invalid inputs** gracefully.  

This project demonstrates **classes, objects, methods, dictionaries, loops, and conditionals**.

---

## 🍨 Features

- **Ice cream flavours** with prices: Chocolate, Vanilla, Strawberry, Mango  
- **Optional toppings**: Sprinkles, Nuts  
- **Cart system**: Add an item to cart, view info  
- **Buy ice cream** option with **bill calculation**  
- Input **validation** for flavour and topping  
- Friendly CLI interface
